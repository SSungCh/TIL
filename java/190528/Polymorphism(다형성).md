### Polymorphism(다형성)

의미는 같지만 실질적으로 표현되는게 다르다, 클래스가 하나의 메시지에 대해 각 클래스가 고유의 방법으로 응답할 수 있는 능력,

> 오버로딩 : 기존에없는 새로운 메서드를 정의하는 것
>
> 오버라이딩 :  상속받은 메서드의 내용을 변경하는 것
